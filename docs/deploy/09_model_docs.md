# Deploying Model Documentation

model specific documentation can be added to a model. If provided, a new page will appear when selecting a scenario.

![Alt text](/deploy/docs_page.png)



### writting doc

doc is written as .md files and supports png and gif.

The page is the same as **"Results Pictures"**, so uploading a docs.md with all its associated pictures can be used for testing.
